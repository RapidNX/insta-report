## Getting Started

### Prerequisites

Clone the Repository
* Install Python3
  ```sh
  $ apt install python3
  ```

### Installation

_Install Insta-Report_


2. Clone the repo
   ```sh
    $ git clone https://github.com/Aqtic0l/Insta-Report.git
   ```
4. Go to the Folder
    ```sh
    $ cd Insta-Report
    ```   
7. Install the Requirements.txt
   ```sh
   $ pip install -r requirements.txt
   ```
4. Run the `insta.py`
   ```sh
   python3 insta.py
   ```
